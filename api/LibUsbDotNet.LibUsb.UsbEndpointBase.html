Transfer(ReadOnlySpan<byte>, int, out int)

Synchronous bulk/interrupt transfer function for read-only buffers.

public virtual Error Transfer(ReadOnlySpan<byte> readOnlyBuffer, int timeout, out int transferLength)

Parameters

readOnlyBuffer ReadOnlySpan<byte>

An ReadOnlySpan<T> to a caller-allocated read-only buffer.

timeout int

Maximum time to wait for the transfer to complete.

transferLength int

Number of bytes actually transferred.

Returns

Error

True on success.

Remarks

This method passes a data buffer (readOnlyBuffer) to the native API, but unlike the Transfer(Span<byte>, int, out int), it ignores the contents of the buffer without copying them, even if data has been written to it. If you expect to receive data from the native API and have it written to the original buffer, use the Transfer(Span<byte>, int, out int) method.
